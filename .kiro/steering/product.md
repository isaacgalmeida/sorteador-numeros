# Product Overview

**Sorteador de NÃºmeros por Grupo** (Number Lottery by Group) is a web-based lottery application that allows users to create multiple groups and randomly draw numbers from each group.

## Core Features

- Create multiple lottery groups with custom names and icons
- Define number pools for each group
- Specify how many numbers to draw per group
- Animated card flip reveal with countdown timer
- History tracking of all drawn numbers
- Share/copy results functionality
- Persistent data storage using localStorage
- Mobile-first responsive design

## User Experience

- Portuguese language interface (pt-BR)
- Visual feedback with celebrations and notifications
- Touch-friendly interactions for mobile devices
- Accessibility features including keyboard shortcuts
- Offline-capable single-page application
